\paragraph{Limitations.}
\todo{(i) Backend coverage limited to \amem{} / \graphiti{};
extending to MemOS / MemMachine is engineering. (ii) Adversary that
can both write the snapshot \emph{and} read the secret key trivially
forges; we assume the key is not exfiltrated. (iii) cat-5
adversarial questions remain hard; F1 lifted to $\approx 0.4$ after
the loader fix but still below paper-level for some backends. (iv)
\texttt{contradiction\_rate} / merge / delete / stale-memory / temporal
consistency need \longmemeval{} ground-truth labels and are not
reported in this draft.}

\paragraph{Future work.}
\todo{(i) Larger $K$ — current self-reported enumeration is $K=4$;
$K{=}8$ doubles per-decision capacity but cost / quality trade-off
needs study. (ii) Online attestation for streaming agents.
(iii) Cross-session anchor chaining for long-running deployments.}
