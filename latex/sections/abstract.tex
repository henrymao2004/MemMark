\begin{abstract}
\memmark{} is a watermark for long-term agent memory systems that
embeds attribution into the backend's own latent state-evolution
decisions. The core challenge is forensic: when a memory snapshot leaks,
is distilled, or migrates across systems, the visible action trajectory
and user-facing output are usually gone, so action-layer and text-layer
watermarks no longer apply. Our method elicits multiple admissible
candidates at each internal LLM call, keyed-selects among them with a
distribution-preserving sampler, and commits the resulting decisions
into an audit trace that is verifiable even when the external log is
missing.

Across \amem{} and \graphiti{} on \locomo{}, \memmark{} preserves
utility within statistical noise of the unwatermarked baseline,
achieves full snapshot-only attribution,
and remains robust under a broad set of memory-lifecycle attacks.
Taken together, these results show that memory attribution is feasible
without sacrificing the utility that makes long-term memory worth
using in the first place.

Code, audit traces, and benchmarks are released.
\end{abstract}
