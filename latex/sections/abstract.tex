\todo{Abstract: provenance-as-fields is honest-writer; attacker who
holds the memory snapshot can rewrite every metadata field.
\memmark{} embeds attribution into the memory system's own latent
evolve decisions (which existing memory to update / link / how to
phrase the new memory) via PRF-keyed sampling on the
distribution-preserving \agentmark{} sampler, and binds those
decisions into a commitment + Merkle log whose anchor lives inside
the memory snapshot. Headline: In-Record Attribution Verification
(R3) decodes the writer's key from snapshot alone — no external
audit store. Evaluated on \locomo{} / \longmemeval{} / \mab{} across
\amem{} and \graphiti{}, \memmark{} preserves utility (RQ1, F1
within statistical noise), embeds non-trivial capacity (RQ2,
$\approx$0.05 bits/decision), recovers bits at $\sim$1.0 in R1/R3
(RQ3), degrades smoothly under 9 memory-specific attacks (RQ4), and
introduces no integrity regression (RQ5).}
