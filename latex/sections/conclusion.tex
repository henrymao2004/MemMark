\section{Conclusion}
\label{sec:conclusion}

\memmark{} shows that long-term agent memory can be watermarked at the
level where the memory backend decides how state evolves, rather than
at the level of visible actions or final text. Across \amem{} and
\graphiti{} on \locomo{}, the method preserves utility within
statistical noise of the unwatermarked baseline, achieves full
snapshot-only attribution in the R3 setting, and remains
robust under a broad set of memory-lifecycle attacks. Taken together,
these results suggest that attribution for memory systems is feasible
without sacrificing the utility that makes memory worth using in the
first place.

Code, audit traces, and benchmarks are released.
